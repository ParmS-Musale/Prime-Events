# Prime Events | Luxury Event Management & Premium Decor

A stunning, production-ready website for a high-end luxury event management company, built with Next.js 15+, React 19, Tailwind CSS v4, Framer Motion, and GSAP.

## Features

- 🎨 **Cinematic Luxury Design**: Glassmorphic UI, Champagne Gold gradients, Obsidian black background.
- ⚡ **Animations**: Smooth scroll reveal animations with Framer Motion, and complex timeline typography reveals with GSAP.
- 📱 **Mobile Responsive**: Fully fluid typography and responsive grid structures.
- 🧭 **Sticky Navigation**: Glassmorphic top navigation with active scroll-spy underlines and mobile drawer.
- 💬 **WhatsApp Integration**: Floating action button and integrated contact form directly bridging to WhatsApp.
- 📷 **Portfolio Lightbox**: Masonry gallery with filtering and responsive lightbox overlay.
- 🚀 **Performance Optimized**: Next.js App Router, advanced font optimization (Playfair Display & Inter), and image remote patterns setup.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion, GSAP (`@gsap/react`)
- **Icons**: Lucide React + custom inline SVGs
- **Deployment**: Ready for Vercel / Netlify

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: Contains the root layout and main page assembly.
- `src/components`: Contains all modular UI sections (`HeroSection`, `AboutSection`, `ServicesSection`, `PortfolioSection`, etc.)
- `src/app/globals.css`: Tailwind configuration and global custom CSS utilities (like `.glass-panel` and `.gold-gradient-text`).
